# humandesign_nodejs

基本上這個專案是應朋友邀來做的實驗性project  

### 專案目標
1. 能成功抓取網頁資料
2. 能依據class來分析所需要的資訊
3. 將抓取到的資料塞到雲端google excel

### 專案技術
1. nodejs
2. npm相關爬蟲套件
3. google 雲端相關api
